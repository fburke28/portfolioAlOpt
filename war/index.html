Stock Evaluation Flow

1. Check and seed companies if necessary

2. Retrieve latest company data from finance service. Load as - sector : List of companies

3. For each account rate users based on performance of stock ratings

4. Rate stocks based on performance of stock and updated user rating

App Features

Design

Fluid design working across Android devices. Optimized for performance with user interaction driving required data loads from server.

Social

User interaction drives stock evaluation. Each user has an individual rating that is computed by the system based on the performance of their stock ratings. The app provides a top 5 user ranking to promote competition among users and drive better community collaboration.

Algorithm

Each stock is given a daily evaluation based on the latest financial reference data availale. Each rating a user gives to a stock is taken into account when evaluating a stock's rating. The impact of a user's stock rating is linked to their individual system rating. If a user has a high rating their stock ratings will be given a much larger consideration then a user with a poor individual rating.

About

The social stock app is available on the Android play store for download. The app has a local data store of financial reference data. This data gets updated along with systematically computed data at the end of every business day.

The SocialStock app interacts with a Google App Engine backend server. This server has a cold startup which means it may periodically take a number of seconds to service loading data from the server to the client.
