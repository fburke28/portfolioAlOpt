API layer for stock evaluator

Opt cycle flow

  1. Check and seed companies if necessary.
  2. Retrieve latest company data from finance service. Load as - sector : List of companies
  3. For each account rate users based on performance of stock ratings
  4. Rate stocks based on performance of stock and updated user rating

Company APIs

  • List of sector names
  • Retrieve list of companies per sector -> symbol, name
  • Retrieve company details by symbol
  • Persist stock rating, comment

Account APIs

  • Retrieve account for userName
  • Retrieve all accounts
  • Retrieve average user account rating
  • Retrieve top ten rated users
  • Retrieve current users overall ranking

Stock Rating APIs

  • Top ten rated stocks
  • Top five highest rating gain
  • Top five highest rating loss
  • Retrieve ratings per sector
